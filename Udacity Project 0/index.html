Mark-up

Mark-Up is the way the code is written. It contains the opening and closing tags with the contents between the tags. This is called the Element.
Example: <Opening Tag>Contents<Closing Tag>

Some of the Tag examples look like this:
< b > = bold
< em > = emphasis/italics

White Space

In HTML, all lines of text are written on one line unless they are forced onto multiple lines using Break Tags or Paragraph tags

-Break Tag= < br > :This tag start text on a new line
-Paragraph Tag= < pr >: This Tag needs an opening and a closing tag. This tag has content inbetween the tags.

Inline vs Block

Inline tags break line ands start new ones.
Block tags create an "invisible box" around text which has depth and height.
Example Inline tags = Bold Tags, Emphasis tags and Image tags.
Example of Block Tags= Paragraph tag.

SPAN/DIV
-Span = Inline
